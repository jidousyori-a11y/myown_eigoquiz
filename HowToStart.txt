英単語クイズ（AI例文リクエスト機能つき）起動方法
====================================================

前提
----
・このPCには Node.js がインストール済みです。
・Gemini APIキーは、このPCのユーザー環境変数 GEMINI_API_KEY に
  永続的に登録済みです（PCを再起動しても消えません／再設定不要）。

起動手順
--------
1. PowerShell を開く

2. このフォルダに移動する
     cd "r:\PUBTEMP\FY26\AI_Experiment\英単語"

3. サーバーを起動する
     node server.js

   次のように表示されれば起動成功です。
     英単語クイズ: http://localhost:51735 で起動しました
     GEMINI_API_KEY: 検出済み

4. ブラウザで以下を開く
     http://localhost:51735

   ※ ポート番号 51735 は、他のアプリ（「情報カード」など）と
     ぶつからないように選んだ番号です。
     もし将来このポートも他のアプリと衝突した場合は、
     PowerShell で PORT を指定して起動できます。
       $env:PORT = "任意の空いている番号"; node server.js

終了方法
--------
・サーバーを起動した PowerShell のウィンドウで Ctrl + C を押す
  （ウィンドウを閉じても停止します）

補足
----
・「✨ 例文をAIにリクエスト」ボタンは、この node server.js を
  起動しているときだけ使えます。
・GitHub Pages 上のバージョンにはこの機能はありません
  （クイズ本体は通常どおり動作します）。
・もし「GEMINI_API_KEY: 未設定」と表示された場合は、
  環境変数が消えている可能性があります。PowerShell で
     setx GEMINI_API_KEY "APIキーの値"
  を実行し、PowerShell を開き直してから再度 node server.js を
  実行してください。
